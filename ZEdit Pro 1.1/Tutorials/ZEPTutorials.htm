ZEdit Pro 1.1 Tutorials


Tutorial Documents

The Tutorial Set for ZEdit Pro 1.1 consists of 5 tutorials, each covering different aspects of model repair, coloring and print preparation.

The Tutorial documents and the geometry files for the tutorials examples are found in the "Tutorials" folder and its sub folders ("Tutorial 1", "Tutorial 2" etc.)  in the ZEdit Pro installation directory (Default: "C:\Program Files\ZEdit Pro 1.1").

Tutorial document (PDF) Topics Estimated time requirement
Tutorial 1 - Lamp Basic automatic repair and coloring. 60 minutes
Tutorial 2 - Black Bass More advanced repair with surface joining and adding thickness. 30 minutes
Tutorial 3 - Annie Repairing and hollowing, adding drain holes. 20 minutes
Tutorial 4 - Cow Splitting and connecting. 20 minutes
Tutorial 5 - Explorer Repair with offsetting, repairing and joining surfaces. 20 minutes

If you are a first timer with ZEdit Pro we recommend you to go through the tutorials in the given order. Especially the Tutorial 1 includes information on the basic use (auto repair, viewing, selecting, painting, texturing) and different concepts (modes, windows etc.) in the software.

Video Tutorials

ZEdit Pro 1.1 Video Tutorial set consists of 14 different tutorials. The first one is a general introduction to the ZEdit Pro user interface. The other tutorials each show you how to use one specific feature of the program.

The Video Tutorial files are found in the "Tutorials" folder and its sub folders ("01_ZEdit Pro Interface", "02_Auto Repair" etc.) in the ZEdit Pro installation directory (Default: "C:\Program Files\ZEdit Pro 1.1").

Video TutorialTopics
01_ZEdit Pro InterfaceGeneral view of the user interface of ZEdit Pro.
02_Auto RepairFixing model errors with the Auto Repair feature.
03_Fill GapUsing the Fill Gap feature to fix a specific type of error in a model.
04_Join GapsUsing the Join Gaps feature to fix a specific type of error in a model.
05_OffsetCreating thickness to surfaces using the Offset feature.
06_PaintingChanging the color of various parts of a model using the Painting feature.
07_Add Engineering TemplateDefining and adding an engineering template to a model.
08_Add Text AnnotationDefining and adding a text annotation to a model.
09_Add TextureAdding a texture to a model.
10_Split PartCutting a model in two using the Split feature.
11_Add Pins and HolesAdding Pins and Holes for connecting a splitted model.
12_Hollow PartMaking a model hollow.
13_Add Drain HoleCreating a Drain hole to a hollow model.
14_Extrude SurfaceAdding thickness to surfaces using the Extrude Surface feature.

Please visit Z Corp’s 3DP User Website www.3dpuser.com for the latest ZEdit Pro Video Tutorials.

Note: The Tutorials were originally created for the ZEdit Pro version 1.0, but they are still applicable for the version 1.1. The features explained in the Tutorials have not changed between versions 1.0 and 1.1.
