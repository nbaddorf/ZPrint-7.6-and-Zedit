Tips:

ZEdit Pro offers two color modes, "3D color printer" and "Repair" (set in View>Toolbars>Display Options>Colors). It is more efficient to do the repair in the Repair color mode which shows the error triangles in red color vs. correct pink color.

You can fit a single shell on the graphics rea by doulbe clickign on it in the Model Tree.

You can select shells and surfaces from teh grahpics area:
  • selection mode is entered by pressing ESC
  • shells are selected with ALT + left mouse
  • surfaces and gaps are selected with left mouse
  • multiple selections is created by pressing CTRL simultaneously
You can graphically select gaps (red lines) behind surfaces by Showing curves through surfaces (View>Toolbars>Display Options Window>Viewing). Each selection in the graphics area is highlighted in the Model Tree and sourrounded by bounding box (green).
