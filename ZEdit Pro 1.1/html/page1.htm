Introduction to File Fixing                    

Optional Tools: ZEditPro allows you to transfer your 3D files directly to Z Corp 3D Color printers through the ZPR file format. Optional Tools from DeskArtes allow you to use other file formats, like STL, IGES, CATIA and STEP, to access any 3D CAD or 3D Printing system. Please see more at the ZEdit Pro registration.

ZEdit Pro has been designed to aid in the fixing of 3D model files for the output to Z Corporation’s Monochrome & Color 3D rapid prototyping printers.  In addition to fixing 3D model files, ZEdit Pro also allows for the coloring and texturing of 3D models – a robust tool for printing great looking monochrome, color and textured 3D models.

File Fixing – What is it?

In order to print consistently great models, a 3D model’s geometry must be in a “correct” state for optimal printer output.  A typical 3D model often consists of multiple shells (parts), and each shell consists of a number of triangles (polygons), often referred to as a triangle (or polygonal) mesh.

A "correct" 3D model file (and its associated shells) will have a "solid" triangle topology; a topology in which each triangle has exactly one neighboring triangle at each of its edges. All shells must be complete solids (without holes).

Error conditions such as inverted normals (inverted polygons), gaps (holes), overlapping triangles, duplicate triangles, edge error triangles and geometry with very thin features (features which are too thin to successfully print) need to be addressed and “corrected”, if necessary.  This is where ZEdit Pro aids in fixing these common error conditions.

More information about the different error types can be found at the Online Help, book "Handling Faceted Models", page "Triangle Errors".  

 

How ZEdit Pro works?

ZEdit Pro offers extremely easy to use tools for file fixing and coloring. After starting the software and inputting a file the user selects the preferred operation mode: Fix Model, Paint and Texture Model, Print Preparation or View Model. In each mode the most efficient set of tools is presented to the user through a separate Tools Window show automatically for each mode. With Fix Model and Print Preparation modes the model structure is presented in the Model Tree Window for easy access to shells, surfaces and gaps. 

Fix Model mode provides tools for fixing the model efficiently. The fixing contains typically three steps: verification, automatic fixing and in rare cases some manual work for more problematic error cases. The most straightforward way is to input the model, select the Fix Model mode  when the part is automatically analyzed for errors. If errors are found automatic repair is prompted, pressing yes will run the automatic repair. After the repair the Model Tree window displays each error shell with red icon. Automatically opened Fix Model tab includes the Help text window to aid the user to select correct commands for the repair work at any stage of the repair process. When the shell icon turns green it can be printed. 

Paint and Texture Model mode displays an extensive set of tools to divide a model into surfaces, paint shells and surfaces as well as to texture them. When entering the Paint and Texture mode the model is verified (if not already done) to make sure there are no errors and to subdivide the part into more manageable surfaces for painting and texturing. Model can be painted in shell, surface and triangle level and texture images in different formats can be superimposed to shells and surfaces. 

Print Preparation mode tools allow more flexible building of a part. Print Preparation offers tools to split, connect and hollow a repaired model. Splitting the model may be necessary to fit the part in the build volume and hollowing combined with drain hole minimize the use of the powder material when printing parts with large volume. 

In View Model mode no verification or repair is prompted automatically. The user can view and select any commands freely from the menus, like transformations or repair commands. The View Model mode is recommended for more advanced users, normally you should start your part preparation with the Fix Model mode. However, the user can change the operation mode any time through the operation mode window.


Where to start with ZEdit Pro?


The easiest way to start using ZEdit Pro software is to go through the Tutorials in the "Tutorials" directory under the ZEdit Pro installation. There are 5 separate tutorials each concentrating on different aspects of the model repair, coloring and manipulation for 3D Printing. 

Or you can also just input models and give the automatic repair a try. When finished, the model structure and potentially remaining errors are shown in the Model Tree Window. Selecting shells in the Model Tree will bring up repair help in the Fix Model window for most efficient repair work. 

When the model is repaired it can be painted or textured with the Paint and Texture Model mode. Also, after the repair model can be manipulated to fit into the build volume or made hollow to save material in the Print Preparation mode.

To get more information on the use of the Model Tree please have a look at the Online Help, book "Handling Faceted Models", page "Using Model Tree". More information on the repair workflow can be found in the "Repair Workflow" page.
